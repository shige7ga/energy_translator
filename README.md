# エネルギー翻訳機
## 📝 概要
エネルギー量（kcal）を、日常生活でイメージしやすい指標に変換するWebアプリです。
数値だけでは分かりづらいエネルギー量を、「運動時間」「電気代」「食べ物」などに置き換えることで、直感的に理解できるようにします。

🔗 https://energy-translator-gxjt.onrender.com/

---

## 🎯 作成背景
エネルギー量（kcal）は数値としては理解できても、「実際にどれくらいなのか」を直感的に把握するのが難しいと感じていました。私自身、職場で燃料のエネルギー計算を扱う中で、数値だけでは具体的なイメージが湧かず、不便さを感じていました。

既存サービスでは、kcalをkWhなど別の単位に変換するものは多いものの、「人がイメージしやすい形」に変換するサービスは少ないと感じました。そこで、エネルギー量をより身近な指標に変換し、誰でも直感的に理解できるツールとして本アプリを開発しました。

---

## 🚀 主な機能
入力したエネルギー量（kcal）を、以下のような指標に変換します。

- 🏃‍♂️ 運動量（例：ランニング○分）
- 💡 電気代（例：○円分）
- 🍙 食べ物（例：おにぎり○個分）
